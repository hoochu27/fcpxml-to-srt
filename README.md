# 🎬 FCPXML to SRT Converter
### Final Cut Pro Titles to Standard Subtitles (.srt)
[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_svg)](https://fcpxml-to-srt.streamlit.app)

[English] This is a simple web-based tool to convert Final Cut Pro XML (.fcpxml) files into standard SRT subtitle files. Built with Python and Streamlit.

[한국어] 파이널컷 프로의 XML(.fcpxml) 파일을 표준 자막 파일(.srt)로 변환해주는 웹 도구입니다. 파이썬과 Streamlit으로 제작되었습니다.

## 📖 How to Use / 사용 방법

### In Final Cut Pro
1. Select your project/sequence.
2. Go to **File > Export XML...**
3. Save the file (e.g., `project.fcpxml`).

### In this Web Tool
1. Upload your `.fcpxml` or `.xml` file.
2. Click the **Download SRT** button.
3. Import the `.srt` file into your video or YouTube.

### 파이널컷 프로에서
1. 프로젝트 또는 시퀀스를 선택합니다.
2. **File > Export XML...** 메뉴를 클릭합니다.
3. 파일을 저장합니다 (예: `project.fcpxml`).

### 웹 도구에서
1. `.fcpxml` 또는 `.xml` 파일을 업로드합니다.
2. **SRT 파일 다운로드** 버튼을 누릅니다.
3. 생성된 `.srt` 파일을 영상 편집이나 유튜브 자막으로 활용하세요.
